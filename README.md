### Getting Started

#### 1) Clone project

`git clone git:gitlab.com/friendly-robot/righton.git`



#### 2) Install node modules

`npm install`

#### 3) Install on device

`npm run ios`
`npm run android`
  
#### 4) Start your app

`npm start`
